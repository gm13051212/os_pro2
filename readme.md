## How to use

```
1. sudo ./compile.sh

Open a console for running master program:
2.1 cd user_program/
2.2 ./master 1 ../input/1.in mmap

Open another console for running slave program:
3.1 cd user_program/
3.2 ./slave 1 ../output/1.out mmap 127.0.0.1 
```
